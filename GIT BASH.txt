GIT BASH
COMANDOS BASH:
- LS -> LISTA O ARQUIVOS DENTRO DO DIRETÓRIO;
- CD -> MUDA DE DIRETÓRIO;
- PWD -> MOSTRA O CAMINHO DO DIRETÓRIO ONDE ESTOU;
- LS -A -> EXIBE OS ARQUIVOS OCULTOS;
- CD .. -> RETORNA PARA A PASTA MÃE DO DIRETÓRIO;
- MKDIR 'NOME' -> CRIA UM DIRETÓRIO;
- CAT -> EXIBE CONTEÚDO DO ARQUIVO;
- RM -> DELETA ARQUIVO OU DIRETÓRIO;
- RM -RF -> DELETA DE FORMA RECURSIVA E FORÇADA TODOS OS ARQUIVOS OU DIRETÓRIOS;

COMANDOS GIT
- GIT INIT . -> CRIA UM REP GIT NA PASTA ATUAL;
- GIT STATUS -> EXIBE O STATUS DO TRABALHO;
- GIT ADD 'NOME DO ARQUIVO"-> SELECIONA O ARQUIVO PARA O COMMIT;
- GIT COMMIT -M "MSG" -> FAZ COMMIT NO ARQUIVO;
- GIT LOG -> EXIBE O HISTORICO DE COMMIT;
-  NANO "NOME DO ARQUIVO" -> ABRE O EDITOR DE TEXTO DENTRO DO TERMINAL, CTRL+O+ PARA SALVAR, CTRL+X+ENTER PARA SAIR DO EDITOR;
